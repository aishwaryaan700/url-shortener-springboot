# URL Shortener Backend (Spring Boot)

A simple URL Shortener backend application built using Spring Boot and MySQL.  
The system converts long URLs into short links and redirects users to the original URL when accessed.

##  Features

- Generate short URLs from long URLs
- Redirect users to the original URL
- Track number of clicks for each short link
- API to view analytics/statistics
- RESTful API design

##  Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman (API testing)
